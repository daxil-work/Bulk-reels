# Bulk Reel Generation

Frontend-only reel workbench (Vite + React + Tailwind v4).

## Run the app

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (e.g. `http://localhost:5173`).

**Do not open `Reel.html` directly in the browser** — that is the old CDN prototype without Tailwind. Always use `npm run dev`.

## Build

```bash
npm run build
npm run preview
```
